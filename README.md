# Next Js + Tailwind CSS | App 

### installed process

* yarn create next-app [yourProjectName]
* yarn add -D tailwindcss postcss autoprefixer

* npx tailwindcss init -p

```
content: [
    "./pages/**/*.{js,ts,jsx,tsx}",
    "./components/**/*.{js,ts,jsx,tsx}",
],
```
